## 1. 用户模块
- 用例：重复注册同名用户。
  - 预期：注册失败，提示用户名存在。
- 用例：非法用户名（含空格/特殊字符）。
  - 预期：拒绝注册。
- 用例：密码缺少数字或字母。
  - 预期：拒绝注册。
- 用例：错误密码登录。
  - 预期：登录失败并记录日志。

- 用例：普通用户首次投票。

  - 预期：拒绝。
- 用例：同 IP 不同用户重复投票。
  - 预期：拒绝。
- 用例：投票时间窗口外投票。
  - 预期：拒绝。

## 3. 候选人管理模块
- 用例：管理员新增候选人。
  - 预期：成功。
- 用例：管理员删除不存在候选人。
  - 预期：SQL 无更新，系统不崩溃。

## 4. 统计与日志模块
- 用例：查询统计报表。
